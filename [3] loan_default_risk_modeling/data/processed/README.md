# Processed data

Large binary training artifacts and raw `.rds` files were intentionally left out of this GitHub-ready version.

Recommended approach:
- keep small sample data here if you want a public demo
- keep full `.rds`, `.zip`, and large intermediate files out of GitHub
- use Git LFS only if you truly need to version large binaries

Examples of omitted local-only assets:
- full processed training `.rds`
- archived model outputs
- large zip backups
