
# Customer Churn Ml

## Overview
This project is part of the Analytics Portfolio Suite.  
It demonstrates a complete analytics workflow including data ingestion, exploration, modeling, and evaluation.

## Project Structure
data/ - datasets or references to datasets  
notebooks/ - exploratory analysis notebooks  
src/ - model training and utility code  
results/ - model outputs and metrics  
visualizations/ - charts and reports  

## How to Run

```bash
pip install -r requirements.txt
python src/train_model.py
```

## Methods
Typical workflow:

1. Data ingestion
2. Data preprocessing
3. Feature engineering
4. Model training
5. Model evaluation

## Output
Model metrics and charts are saved in the results/ folder.
